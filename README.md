## Data

This dataset contains the full Lending Club data available from their [*site*](https://www.lendingclub.com/info/download-data.action). There are separate files for accepted and rejected loans. The accepted loans also include the FICO scores, which can only be downloaded when you are signed in to Lending Club and download the data. The data used contain all the Loans of the Lending Club in the period of 2007 - 2018 consisting of > ~1.5M samples with > 150 features.
